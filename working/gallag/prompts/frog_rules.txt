# 개구리 연못 건너기 게임 규칙

## 게임 메커니즘
- 개구리가 연못의 패드를 밟고 건너갑니다
- eliminate_mode에 따라 밟을 수 있는 패드가 달라집니다:
  - **false** (기본): 조건에 맞는 숫자만 밟을 수 있음
  - **true**: 조건에 맞는 숫자는 밟으면 안 됨

## ⚠️ num_range 설정 시 주의사항 (CRITICAL)

**num_range는 반드시 candidates와 호환되어야 합니다!**

- **약수 게임** (candidates: [12, 24, 36])
  - num_range 최소: [[1, 20], [1, 30], [1, 40]]
  - 이유: 12의 약수(1,2,3,4,6,12)가 충분히 포함되어야 함

- **배수 게임** (candidates: [3, 4, 5])
  - num_range 최소: [[1, 20], [1, 30], [1, 40]]
  - 이유: 3의 배수가 최소 5-10개는 있어야 함

- **공약수 게임** (candidates: [[12, 18]])
  - num_range 최소: [[1, 20], [1, 30], [1, 40]]
  - 이유: 공약수(1,2,3,6)가 포함되어야 함

- **최대공약수 게임** (candidates: [[12, 18]], GCD=6)
  - num_range는 **최대공약수를 반드시 포함**
  - 예: [[1, 20], [1, 30], [1, 40]] (6이 포함됨) ✅

- **최소공배수 게임** (candidates: [[4, 6]], LCM=12)
  - num_range는 **최소공배수를 반드시 포함**
  - 예: [[1, 20], [1, 30], [1, 40]] (12가 포함됨) ✅

**절대 하지 말 것:**
- ❌ num_range가 [1, 5]처럼 너무 좁음 → 조건 충족 숫자가 1-2개뿐
- ❌ 최대공약수/최소공배수가 범위 밖 → 게임 바로 종료

## 난이도별 game_rules

사용자가 선택한 난이도에 맞춰 설정하세요:

**쉬움** (초보자 친화적):
- rows: [4, 5, 6]
- num_range: [[1, 20], [1, 30], [1, 40]]
- speed: [30, 40, 50]

**보통** (균형잡힌 난이도):
- rows: [5, 7, 9]
- num_range: [[1, 30], [2, 60], [3, 90]]
- speed: [45, 65, 85]

**어려움** (숙련자용):
- rows: [7, 9, 10]
- num_range: [[1, 50], [1, 80], [1, 100]]
- speed: [60, 80, 100]

각 리스트는 3개 값으로 레벨1→레벨2→레벨3의 점진적 증가를 나타냅니다.

**중요**: 사용자 프롬프트에 난이도 힌트가 포함되어 있다면, 위 가이드를 참고하여 적절한 값을 선택하세요.
**사용자의 추가 요청사항이 있다면 항상 우선시**합니다.
