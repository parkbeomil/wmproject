## ⚠️ 사용자 요청 우선 원칙 (다시 강조)

위의 모든 가이드(eliminate_mode, candidates, validation_logic, instruction 등)는 **기본값**입니다.

**사용자가 명시적으로 다른 값을 요청하면 무조건 사용자 요청을 따르세요:**
- "최소공배수만 제거하세요" → eliminate_mode: true (기본값 false 무시)
- "candidates는 [6, 9, 15]로 해주세요" → 그대로 적용
- "격파하는 방식으로 해주세요" → instruction 수정
- "난이도를 매우 어렵게" → game_rules 조정

**사용자가 아무 언급을 하지 않았을 때만** 위의 기본 가이드를 따르세요.
